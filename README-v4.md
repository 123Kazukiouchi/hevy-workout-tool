# HEVY ワークアウト登録ツール v4.0

## 🆕 v4.0の新機能

### ステップガイド式UI

直感的な4ステップで簡単にワークアウトを登録できます：

#### ステップ1️⃣: APIキー入力
- APIキーを入力
- 「接続テスト」でHEVYと正しく通信できるか確認
- テンプレートデータを事前取得

#### ステップ2️⃣: トレーニングデータ入力
- **方法1（推奨）**: トレーニングノートを貼り付けて自動解析
- **方法2**: 手動でフォーム入力
- エクササイズ一覧からテンプレートID検索
- お気に入り機能で頻繁に使うエクササイズを保存

#### ステップ3️⃣: 内容確認
- 送信するJSONデータをプレビュー
- バリデーションエラーを事前チェック
- 問題があれば詳細を表示

#### ステップ4️⃣: 登録完了
- ワンクリックでHEVYに登録
- 成功/失敗を明確に表示
- エラー時は再試行可能

## 主な改善点

### 📊 視覚的なステップインジケーター
- 現在のステップが一目でわかる
- 完了したステップは緑色のチェックマーク
- 進行中のステップは青色でハイライト

### ✅ ステータスバッジ
- 各カードの完了状態を表示
- 「入力待ち」「入力中」「完了」の3状態

### 🔒 ステップロック
- 前のステップが完了しないと次に進めない
- 入力漏れを防止

### 📱 レスポンシブデザイン
- モバイルでも使いやすい
- カードの折りたたみ機能

## 使用方法

1. **`hevy-workout-v4.0-wizard.html`** をブラウザで開く
2. ステップ1でAPIキーを入力し、「接続テスト」をクリック
3. ステップ2でトレーニングデータを入力（手動または自動）
4. ステップ3で内容を確認
5. ステップ4で登録完了！
