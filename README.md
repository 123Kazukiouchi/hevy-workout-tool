> 日本語のトレーニングデータを貼り付けるだけで、HEVY APIを使って過去・未来のワークアウトを自動登録できる静的Webアプリケーション

## 🚀 クイックスタート

1. **[index.htmlをダウンロード](index.html)** または **[GitHub Pages でデモを試す](#)** 
2. ブラウザで開く（インターネット接続必須）
3. HEVY APIキーを入力
4. トレーニングデータを貼り付けて自動登録！

## 📋 プロジェクト概要

日本語のトレーニングデータ（日付・種目名・セット情報）を貼り付けるだけで、HEVY APIを使ってワークアウトを自動登録できるWebツールです。

### 🎯 主な特徴

- ✅ **完全静的Webアプリ** - サーバー不要、ブラウザだけで動作
- ✅ **日本語完全対応** - 日本語の種目名を自動翻訳
- ✅ **HEVYエクスポート形式対応** - アプリからコピペするだけ
- ✅ **機械学習機能** - 使うほど賢くなる自動マッチング
- ✅ **過去＆未来対応** - 過去のログも、未来の予定も登録可能
- ✅ **プライバシー重視** - APIキーはブラウザのLocalStorageに保存

**最新アップデート（v6.3）：** ⏰ **未来のトレーニング登録機能追加！** クイック日時設定ボタン（明日、明後日、来週など）で、トレーニング予定を簡単に登録できるようになりました。

## 🚀 使い方

### **Step 1: API設定**
1. **初回のみ**: HEVY APIキーを入力
   - 入力後、LocalStorageに自動保存されます
   - 🔒保存済みマークが表示されます
2. **初回のみ**: 「🔍 接続テスト」ボタンをクリック
   - 全529テンプレートが自動取得されます
   - テンプレートもLocalStorageに自動保存されます
3. **2回目以降**: ページを開くと自動的に復元されます
   - APIキーとテンプレート両方が自動復元
   - 即座に「次へ →」で利用開始可能
   - テンプレート取得日時が表示されます
4. **更新**: 「🔄 テンプレートを再取得」ボタンで最新化可能
5. **セキュリティ**: 「🗑 APIキーをクリア」ボタンで削除可能

### **Step 2: データ解析**
1. ワークアウト日時を入力（例: 2025-01-15 18:30）
2. 日本語トレーニングデータを貼り付け：

   **方法1: 従来形式（コロン区切り）**
   ```
   ハンギング・ニーレイズ
   10 / 自重
   10 / 自重
   
   ケーブル・ウッドチョップ
   10 / 20.4kg
   10 / 20.4kg
   ```

   **方法2: HEVYエクスポート形式（v6.1 NEW!）**
   ```
   トレーニング記録 - 8月29日

   サマリー
   • トレーニング数: 10種目
   • 所要時間: 1時間 15分

   Exercise Details

   1. バーベル・スクワット
   • レベル: LV 4
   • 合計重量: 1,200kg

   セット詳細:
   • 80kg × 5回
   • 82.5kg × 5回
   • 85kg × 5回

   2. チンニング
   セット詳細:
   • 10回
   • 9回
   • 8回
   ```

3. 「データを解析」ボタンをクリック
4. 自動的にテンプレートIDがマッチング（学習済みは優先適用）
5. 候補が表示された場合、クリックで選択

### **Step 3: 送信**
1. 解析結果を確認
2. 「HEVYに送信」ボタンをクリック
3. ワークアウトが登録されます
