# クイックスタートガイド

このガイドでは、HEVY ワークアウト登録ツールを**最速で使い始める方法**を説明します。

---

## ⚡ 3ステップで開始

### **1. ファイルを開く**

```bash
# ダウンロードした場合
index.html をダブルクリック

# または、GitHub Pages版を使用
https://yourusername.github.io/hevy-workout-tool/
```

### **2. APIキーを設定**

1. [HEVY公式サイト](https://www.hevyapp.com/)でアカウント作成
2. API設定からAPIキーを取得
3. ツールの「Step 1」でAPIキーを入力
4. 「🔍 接続テスト」をクリック

### **3. トレーニングを登録**

**方法A: 自動解析（推奨）**
```
HEVYアプリからトレーニングをコピー
↓
ツールの「トレーニングデータ貼り付け」欄にペースト
↓
「📥 自動入力」をクリック
↓
完了！
```

**方法B: 手動入力**
```
「➕ エクササイズを追加」をクリック
↓
種目名・重量・回数を入力
↓
「次へ →」をクリック
↓
完了！
```

## 💡 よくある質問

### **Q: APIキーはどこで取得できますか？**

A: HEVY公式サイト（https://www.hevyapp.com/）のアカウント設定 → API設定から取得できます。

### **Q: APIキーは安全ですか？**

A: はい。APIキーはブラウザのLocalStorageに保存され、外部には送信されません。ただし、共用PCでは使用後にクリアすることをお勧めします。

### **Q: オフラインで使えますか？**

A: 初回アクセス後、一部の機能はオフラインでも使用可能です。ただし、HEVY APIへの送信にはインターネット接続が必要です。

### **Q: 日本語の種目名が認識されません**

A: 翻訳辞書にない種目の場合、手動でテンプレートIDを入力するか、「🔍」ボタンで検索してください。よく使う種目は自動的に学習されます。

### **Q: 過去のトレーニングだけですか？**

A: いいえ！v6.3から**未来のトレーニング予定**も登録できるようになりました。クイック日時設定ボタンで「明日」「来週」なども簡単に設定できます。
